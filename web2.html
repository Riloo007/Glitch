1. Set up a workspace

First thing to do is create a folder to keep your website in. On a computer, create a folder and call it whatever you want. Then, create a file called 'index.html'.

1. Hit ctrl - alt - t to open up the terminal.

2. Navigate to your folder with cd <foldername>

3. Type gedit index.html and save the file.

Next, type in this basic HTML code:

<html>
  <head>
    <title>Glitch Website Tutorial</title>
  </head>
  <body>
    <p>
      Hello Earth!
    </p>
  </body>
</html>

And finally, save the file and open it in your browser.
