The Original Thing

"Fancy Graphics"

This program originated from an Atari Graphics book: "BASIC FUN WITH GRAPHICS" Written in BASIC for the Atari 400 in 1995, here's the original code.


100 GRAPHICS 8:SETCOLOR 1,1,14:COLOR 1
105 SETCOLOR 2,6,0
110 PRINT "INPUT SIDE, ANGLE, INCREMENT"
114 SETCOLOR 2,6,0
120 INPUT R,A,D
130 N=360/A
140 A=A*(3.14159/180)
150 PLOT 100,100
160 X1=100:Y1=100
170 I=0
180 X2=R*COS(A*I)+X1
190 Y2=-R*SIN(A*I)+Y1
200 IF X2<0 OR X2>319 THEN END
210 IF Y2<0 OR Y2>159 THEN END
220 DRAWTO X2,Y2
230 X1=X2:Y1=Y2
240 R=R+D
250 I=I+1
260 GOTO 180

Explaination

I'm still not totally sure how this works, but it effectively uses circle math to calculate the next point in the circle. This next point's location can vary depending on the radius (which is increased each interval), and the angle increment (which is increased each drawing).
